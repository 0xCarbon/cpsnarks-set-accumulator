# accumulator
Cryptographic accumulators in Rust, implemented over a generic group interface. Batteries (RSA and
class group implementations) included!

## Installation
```
# Cargo.toml
[dependencies]
accumulator = { git = "https://github.com/cambrian/accumulator.git" }`
```
